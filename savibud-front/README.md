# Savibud (Front-End)

Personal application to handle different aspects:
- Personal Finances
- Meal planning

This is the Front End part of the project.

## 🛠 Tech Stack

This project is powered by:

- ⚡️ [Vite](https://vitejs.dev/) — the ultra-fast build tool that makes development feel instant
- 🧠 [TypeScript](https://www.typescriptlang.org/) — because types are friends, not bugs
- ⚛️ [React](https://reactjs.org/) — for building delightful user interfaces
- Ready to be a Progressive Web App (PWA) if you want it to be 😉

## 🚀 Getting Started

Clone the repo and run:

```bash
npm install
npm run dev
```
